## PROCEDIMIENTO GENERAL DE VENTA

1. Cliente solicita producto
2. Buscar en Smart POS: Catálogo > [Categoría] > [Producto]
3. Seleccionar valor/duración
4. Cobrar al cliente (efectivo o tarjeta)
5. El sistema genera el PIN/código automáticamente
6. Imprimir ticket con el código para el cliente
7. Comisión se abona instantáneamente al saldo

## NOTAS IMPORTANTES
- Los PINs tienen validez de 12 meses desde la activación
- No se admiten devoluciones de productos digitales una vez activados
- En caso de PIN no válido, abrir incidencia con código de transacción
- Verificar siempre que el cliente entiende cómo canjear el código
