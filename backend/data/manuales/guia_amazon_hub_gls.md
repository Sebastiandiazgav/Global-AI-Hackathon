# Guía Operativa: Amazon Hub Counter y GLS Point

## Introducción
Tu punto de venta forma parte de la red de HUBs de recogida de paquetería más grande de España. Al asociarte con Amazon Hub Counter y GLS Point, tu establecimiento se convierte en un punto de recogida seguro para clientes de e-commerce.

## Beneficios para el Punto de Venta
- **Comisión por paquete**: Amazon 0.30€ | GLS 0.25€ | SEUR 0.25€
- **Tráfico adicional**: Clientes que vienen a recoger paquetes compran otros productos
- **Imagen de marca**: Asociación con marcas líderes del e-commerce
- **Sin inversión**: Solo necesitas espacio de almacenamiento y tu Smart POS

## Requisitos del Establecimiento
- Espacio mínimo de almacenamiento: 2m² dedicados
- Horario mínimo: 8 horas/día, 5 días/semana
- Terminal Smart POS o dispositivo Android con cámara
- Conexión a internet estable
- Zona segura y seca para los paquetes

## RECEPCIÓN DE PAQUETES

### Procedimiento Estándar

#### 1. Llegada del Transportista
- El repartidor llega con los paquetes asignados a tu punto
- Verificar que el número de bultos coincide con el albarán
- Si hay discrepancia, anotar en el albarán antes de firmar

#### 2. Escaneo de Paquetes
- Abrir app Disashop > Logística > Recepción
- Escanear código de barras de CADA paquete con la cámara del Smart POS
- El sistema confirma cada escaneo con un "beep" y marca verde
- Si un código no se lee, introducir manualmente los últimos 8 dígitos

#### 3. Modo Escaneo Masivo
- Para más de 3 paquetes: activar "Modo Masivo"
- Escanear todos los paquetes seguidos sin confirmar individualmente
- Al finalizar, el sistema muestra resumen: "X paquetes registrados"
- Confirmar el lote completo con un solo toque

#### 4. Almacenamiento
- Colocar paquetes en la zona designada
- Organizar por transportista (Amazon / GLS / SEUR)
- Los paquetes grandes van en la parte inferior
- Mantener códigos de barras visibles para facilitar búsqueda

#### 5. Firma del Albarán
- Firmar en la tablet del repartidor
- El sistema Disashop ya tiene registrados los paquetes
- No es necesario firmar nada adicional en el Smart POS

## ENTREGA DE PAQUETES AL CLIENTE

### Procedimiento de Entrega

#### 1. Identificación del Cliente
El cliente puede identificarse de tres formas:
- **PIN**: Código de 6 dígitos recibido por email/SMS
- **DNI**: Mostrar documento de identidad físico
- **DNI Digital 2026**: Verificación mediante NFC del nuevo DNI

#### 2. Búsqueda del Paquete
- Abrir app Disashop > Logística > Entrega
- Introducir PIN o escanear DNI
- El sistema muestra qué paquete(s) corresponden al cliente
- Indica la ubicación en tu almacén

#### 3. Verificación
- Escanear el código de barras del paquete
- El sistema verifica que coincide con el cliente
- Pantalla verde = OK, proceder con entrega

#### 4. Firma del Cliente
- El cliente firma en la pantalla del Smart POS
- Se genera confirmación automática
- Amazon/GLS reciben notificación instantánea

#### 5. Ticket
- Se imprime ticket de entrega (opcional)
- El cliente recibe email de confirmación automáticamente

## DEVOLUCIONES

### Tipos de Devolución
1. **No recogido (7 días)**: El sistema alerta automáticamente al día 5
2. **Rechazado por cliente**: El cliente no quiere el paquete
3. **Paquete dañado**: Daño visible en el embalaje

### Procedimiento
1. Acceder a Logística > Devoluciones
2. Escanear código del paquete
3. Seleccionar motivo de devolución
4. Imprimir etiqueta de devolución desde Smart POS
5. Pegar etiqueta sobre la etiqueta original
6. Colocar en zona de devoluciones
7. El transportista lo recoge en su próxima visita (24-48h)

### Importante
- Las devoluciones NO generan comisión
- Pero SÍ se registra la gestión (cuenta para métricas del punto)
- Si un paquete está dañado, fotografiar antes de devolver
